# 💻 Uber Ride Analytics Dashboard

![](https://github.com/AAKASH-BONI/Uber-Dashboard/blob/main/Images/Uber.webp).

## Overview

This project presents an interactive Uber Ride Analytics Dashboard built using Microsoft Power BI to analyze ride booking data and extract meaningful business insights. The dashboard provides a comprehensive overview of ride performance, including total completed rides, lost bookings, revenue generated, travel distance, and average trip metrics.

Through dynamic visualizations and filters, users can explore ride trends across different months, analyze revenue by vehicle types, and identify key operational insights such as top pickup and drop locations, customer ratings, and driver performance. The dashboard is designed to help stakeholders understand ride patterns, monitor operational efficiency, and support data-driven decision-making.

The project demonstrates practical applications of data visualization, business intelligence, and analytical storytelling using Power BI.

## Objectives

- To analyze overall ride booking performance.
- To track the number of completed, cancelled, and incomplete rides.
- To monitor key metrics like total revenue, total distance traveled, and average trip distance.
- To understand monthly ride trends and demand patterns.
- To analyze revenue generated by different vehicle types.
- To identify the top pickup and drop locations.
- To evaluate customer and driver ratings for service quality.
- To present clear insights using interactive dashboards created in Microsoft Power BI.

## Dataset

The dataset used in this project represents a simulated ride-booking platform dataset created for analytical and visualization purposes. It contains structured data related to ride bookings, vehicle types, revenue, trip distance, cancellations, and customer as well as driver ratings.

## Key Features

- Interactive Dashboard built using Microsoft Power BI for easy data exploration.
- Overview Page showing important metrics like total completed rides, lost bookings, revenue, and distance traveled.
- Monthly Trend Analysis to track ride bookings and revenue performance over time.
- Revenue by Vehicle Type to understand which vehicle category generates the most revenue.
- Ride Status Analysis showing completed, cancelled, and incomplete rides.
- Top Pickup and Drop Locations to identify high-demand areas.
- Customer and Driver Rating Analysis to measure service quality and user satisfaction.
- Dynamic Filters and Navigation to explore different vehicle types and time periods easily.
